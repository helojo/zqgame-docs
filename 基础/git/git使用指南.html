git

git软件安装和账号配置

官方下载:https://git-scm.com

网盘下载:Git-2.22.0-64-bit.exe

  1. 配置git用户信息

    git config --global user.name “用户名”
git config --global user.email “邮箱地址”

    以上命令中，用户名为邮箱中的用户名(一般为姓名拼音首字母小写缩写)；邮箱地址为自己邮箱。

  2. 生成rsa加密公钥和私钥

    ssh-keygen -t rsa -C "邮箱地址"

    后面的步骤直接回车就可以,如下图所示，加密公钥和私钥就生成了。

  3. 到用户目录下隐藏文件夹 .ssh 下找到 id_rsa.pub 文件，用记事本打开，复制文件中的内容。
